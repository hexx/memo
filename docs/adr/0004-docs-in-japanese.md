# ユーザー向けドキュメントは日本語で書く

本プロジェクトのユーザー向けドキュメント（CONTEXT.md・ADR・design.md）は日本語で書き、コード識別子とファイル名スラグは英語のままにする。UI と README はすでに日本語で、ADR 0003 も日本語で書かれていた一方、CONTEXT.md・ADR 0001/0002・design.md は英語のままでドキュメントの言語が混在していた。プロジェクトの言語を統一するため、既存の英語ドキュメントを日本語化し、用語の正規名は日本語＋英語併記で CONTEXT.md に載せる方針に定めた。今後の新規 ADR やドキュメントも日本語で書く。

## Considered Options

- **英語で統一する**: オープンソースでの外部貢献やツール親和性では有利だが、ユーザー・UI・README が日本語の本プロジェクトでは一貫性のメリットが上回ると判断した。
